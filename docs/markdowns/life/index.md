# 先交学费再梭哈

起初是玩游戏充值方面，总是在新号或者是基础很差的号上直接投入精力、金钱。

但是，往往事与愿违，游戏体验并不理想，最后导致投入的资源打了水漂或者是起步太慢、太差还不如别人投入一点点就能做到的地步。

拿PUBG举例， 刚开始玩自己的号，发现好看的枪皮、工坊衣服等都需要投入大量的精力时间，要不就是去氪金。 自己拿着自己申请的邮箱号去慢慢养了大概3个月，发现还是一塌糊涂。
加上自己忍不住去养，总是一点道具就去抽掉。 

然后呢， 在看直播的过程中，发现可以有小白号，是那些脚本商几年时间内不断的挂机领取奖励，起步非常快， 而且在直播间up主也有很多抽中免单活动， 抽了一次之后发现雀氏快的很！ **这是第一笔学费**

这一笔学费， 省掉了无用的精力投入，见效很快。

以为事情到这里就结束了，但是在加入粉丝群后， 发现还有门道！ **有类似活动的返场、各种白嫖的骚操作， 但是自己已经投入一些不该投入的金钱**， 这是**第二笔学费**
