# 条件分支 if判断

## 语法

`py`语法中， 也是简单的不带括号的形式：

```python
num = 12
if num == 1:
    print("num等于1")
elif num == 2:
    print("num等于2")
else:
    print("num不等于1也不等于2")
```

## 其余注意

可以与`js`对比:

- 区分大小写

- 只有`==`，没有`===`

- `&&` 改为 `and`

- `||` 改为 `or`

- 列表包含判断 改为 `in`

- 非包含判断 改为 `not in`

- 布尔值为 `True` 和 `False`

## if语句

完整版为`if ... elif ... else ...`

## 列表if

:::tip
直接判断列表， 为空返回`False`
:::
