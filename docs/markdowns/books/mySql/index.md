# 数据库及SQL

### 数据库

本书而言，数据库可以想象是一个文件柜，是一个存放数据的物理位置。

**数据库**：保存有组织的数据的容器(一个/一组文件)。 

**表**：某种特定类型数据的结构化清单。 表具有一些特性， 定义了数据如何存储在表中，可以存储什么样的数据， 数据如何分解、各部分信息命名等， 这组描述特性信息就是**所谓的模式(schema)**。

**列**：表中一个字段， 正确的**数据分解**为列很重要，比如城市和州应该分成两列， 而不是放在同一列中。

**数据类型**：每列对应一种数据类型，限制列存储数据类型

**行**：表中的一条记录，别名也可以叫record数据库记录

**主键(primary key)**：唯一标识符，区分表格每一行。  应该总是定义主键，确保操作和管理

> - 每行主键必须不同、每行必须有主键值而不是`null`
> - 主键可以是列的组合，单个值可以不唯一，但是组合必须符合上述规则
> - 主键好习惯： 不更新、不重用、不使用可能会更改值作为主键值

### SQL

structured query language，结构化查询语言

> SQL不是专利语言， 存在一个标准委员会， 但是不同数据库实现会有差异，虽然适用但不可完全移植.
